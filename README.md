## Installation 

```
https://github.com/devjewel01/AI-Voice-Assistant
```
```
cd AI-Voice-Assistant
```
**For Windows users:**

```
pip install -r requirements.txt
```

**For Linux users:**

```
pip3 install -r requirements.txt
```
```
cd src
```
```
python3 main.py
```
